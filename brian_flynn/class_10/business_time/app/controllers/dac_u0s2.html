In this slideshow, we'll explore functions in Excel and the difference between absolute and relative cell referencing.

First, let’s use a formula to discover the total number of sales for the year.

We could add each cell together one by one; however, once again, there’s a better way.

Instead of manually adding each cell, we’ll use a function.

Excel functions are formulas that are built into the software; we'll teach you several of the most useful functions later in this course.

In this case, since we need the total sum of a range of numbers, we’ll use the SUM function.

Functions in Excel must always be followed by parenthesis, where you’ll enter the values to which you want the function to apply.

Instead of entering each value individually, you can just click and drag to select the whole range.

Ranges in Excel are expressed by entering the first value, followed by a colon, then the final value.

Once again, if you compare the cell to the formula bar, you can see that the formula bar still contains the raw data that you entered — =SUM(B4:B15) — while the cell presents the resulting value — 616 items sold.

We can click and drag this function to instantly calculate the sum of the total sales revenue for the year.

Next, let’s add a new column so we can do some additional calculations.

To do so, just right-click a column and select “Insert.”

New columns will always insert to the left of the selected column.

New rows will always insert above the selected row.

We’ll name our new column “Percent of Annual Sales.”

We can click and drag the column head to widen the column and better fit our title.

For each row in this column, we’ll calculate the percent of annual revenue collected.

We can do so by dividing the total revenue for the month divided by the total revenue for the year.

The answer doesn’t look right, does it?

That’s because when we inserted the column, it held the currency formatting from the Average Sales column.

We need to switch the formatting from “currency” to “percentage.”

That looks better!

January accounted for 8% of the annual revenue.

However, if we click and drag our formula to apply it to the rest of the column, we’re left with a bunch of errors.

What happened?

If you take a look at the first error, you might notice something funny in the formula bar: it says =C5/C17 — the second coordinate refers to an empty cell.

This incorrect pattern continues for each row in the column.

The type of cell referencing we’ve been doing so far is known as relative cell referencing. This is the default in Excel.

Relative cell references change based on the relative position of rows and columns.

For example, if you copy the formula =A1+B1 from row 1 to row 2, the formula will become =A2+B2.

If you want a cell reference to remain constant, you have to use absolute cell referencing.

To signify this, you have to place a dollar sign before each coordinate that you would like to stay the same.

For example, if you copy the formula =$A$1+$B$1 from row 1 to row 2, the formula will remain =$A$1+$B$1.

Let’s try this with our sample data.

In this case, we want our first reference — the monthly revenue — to remain relative, but our second reference — the total annual revenue — to be absolute, so we’ll type =C4/$C$16.

If we apply this new formula to the rest of the column, it works perfectly.

And if we compare the data in the second row of this column to the formula bar, we can see that our first reference has changed appropriately and the second has remained constant: <=C5/$C$16.
